Formatters for the main resource tree structure

module OCaml: 
functor (SF : OCamlResSubFormats.SubFormat) -> Format 
  with type data = SF.t
   and type params = ocaml_format_params

This format produces OCaml source code with OCaml submodules for directories and OCaml value definitions for files.
